About

AVR Emulator is a project of two students at the HTL Rankweil for AINF.

Simulation

To start the simulation you can choose between two different kinds. You can simulate it manually. Therefore, if you press the manual button, the next instruction is being executed. If you want to simulate the programm without pressing the manual button for each line, you can simulate it automatically. When you simulate it automatically you can choose between different speeds for the simulation. To stop the simulation you can press the reset button. Then the simulation stops and you can start it again. If there is a syntax error somewhere in your code, an error is displayed when you try to simulate that line. Otherwise a green arrow indicates which line will be simulated next.

Memories

There are 3 types of Memory in the Atmel AtTiny2313 to store Data:

  • Working Register: 32 Byte
  • Static RAM: 128 Byte
  • Peripheral Registers: Access to the internal Modules
The Memories of the Tiny2313

The value can be modified by doubleclicking on a entry.

Ports

The Atmel AtTiny2313 features 18 bi-directional I/O pins, distrubuted over 3 ports:

  • Port A: 3 Pins
  • Port B: 8 Pins
  • Port C: 7 Pins

The direction of each ports is determined by the state of the DDRX register, and also is displayed in the I/O-Ports section of the simulator. When in output Mode, the digital state state (High / Low) of each pin corresponds to the bits in the PORTX register. When in input mode, the state of a pin can be change from within the GUI of the simulator and is reflected by the value of the PINX register.

Keyboard Shortcuts

  • New: Ctrl+N
  • Open: Ctrl+O
  • Save: Ctrl+S
  • Reset: F2
  • Step: F11
  • Auto: F5
